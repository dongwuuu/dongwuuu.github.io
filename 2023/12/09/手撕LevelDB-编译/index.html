手撕LevelDB-编译

仓库下载

git clone git@github.com:google/leveldb.git

编译

  1. cd leveldb
  2. git clone https://github.com/google/googletest.git
  3. git clone https://github.com/google/benchmark.git
  4. mkdir build && cd build
  5. cmake -DCMAKE_BUILD_TYPE=Release .. && cmake –build .

常见问题

third_party/benchmark does not contain a CMakeLists.txt file.

编译时遇上以上报错，简易重新执行第2-3步。
